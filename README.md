# 🏋️ Gym Tyme

Gym Tyme is a simple web application for browsing and booking gym equipment online. It includes a landing page, product sections, and a user booking form.

## 📸 Preview

## 🚀 Features
- Responsive landing page
- Gym equipment showcase
- Booking form with:
    - Name & Email input
    - Gender selection
    - Equipment selection (checkboxes)
- Equipment categories section
- Membership section

## 🛠️ Tech Stack
HTML5
CSS3 (linked via style.css)
Basic frontend UI
